Step 1: Set Up Your Python Environment

First, create a new, empty folder on your computer to hold the project files. This will be your main project directory. All subsequent files and folders will be created inside this one.

Step 2: Create `pyproject.toml`

Inside your project folder, create a file named `pyproject.toml`. This file manages your project's dependencies and tells Python how to package your tool. Copy the content below into this file.

[build-system]
requires = ["setuptools>=42", "wheel"]
build-backend = "setuptools.build_meta"

[project]
name = "url-text-fetcher"
version = "0.1.0"
description = "FastMCP server for URL text fetching"
authors = [{ name="Your Name", email="your_email@example.com" }]
dependencies = [
    "fastmcp",
    "requests",
    "beautifulsoup4",
]

[project.scripts]
url-text-fetcher = "url_text_fetcher.mcp_server:main"

Step 3: Create the MCP Server Script

This Python script contains the actual logic for your tool. It's important to get the folder structure correct. Inside your main project folder, create a new subfolder named `url_text_fetcher`. Then, inside that new subfolder, create the file `mcp_server.py`.

Copy the following code into `mcp_server.py`:

import requests
from bs4 import BeautifulSoup
from mcp.server.fastmcp import FastMCP
from typing import List

mcp = FastMCP("URL Text Fetcher")

@mcp.tool()
def fetch_url_text(url: str) -> str:
    """Download all visible text from a URL."""
    try:
        resp = requests.get(url, timeout=10)
        resp.raise_for_status()
        soup = BeautifulSoup(resp.text, "html.parser")
        return soup.get_text(separator="\\n", strip=True)
    except requests.exceptions.RequestException as e:
        return f"An error occurred while fetching the URL: {e}"

@mcp.tool()
def fetch_page_links(url: str) -> List[str]:
    """Return a list of all links on the page."""
    try:
        resp = requests.get(url, timeout=10)
        resp.raise_for_status()
        soup = BeautifulSoup(resp.text, "html.parser")
        return [a['href'] for a in soup.find_all('a', href=True)]
    except requests.exceptions.RequestException as e:
        return f"An error occurred while fetching the URL: {e}"

def main():
    mcp.run()

if __name__ == "__main__":
    main()

Step 4: Install Your Local MCP Server

Open your terminal or command prompt. Navigate into your project's root folder (the one containing `pyproject.toml`). Then, run the following command to install your server as an editable local package. This allows you to make changes to the script without needing to reinstall.

pip install -e .

Step 5: Register Your Server in LM Studio

Now, you need to tell LM Studio about your new tool.

  1. Open LM Studio and go to the chat window.
  2. At the bottom, find the "Integrations" section and click "Edit mcp.json."
  3. Add the configuration below. This example also includes the Brave Search tool, which you can use by getting a free API key from Brave.
{
  "mcpServers": {
    "url-text-fetcher": {
      "command": "python",
      "args": [
        "-m",
        "url_text_fetcher.mcp_server"
      ]
    },
    "brave-search": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-brave-search"
      ],
      "env": {
        "BRAVE_API_KEY": "put_your_brave_api_key_here"
      }
    }
  }
}

Step 6: Use the Tool in LM Studio

With the `mcp.json` file saved, you can activate your tools.

  1. In the LM Studio chat window, click the "Integrations" button (it looks like a plugin icon üîå).
  2. You should now see both `mcp/url-text-fetcher` and `mcp/brave-search` listed. Select one or both to enable them for the session.
  3. You can now prompt your model to use the tool.
